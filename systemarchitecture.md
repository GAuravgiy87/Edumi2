## Request Flows

```
  WEB REQUEST
  ──────────────────────────────────────────────────────────────────────────────
  Browser ──► Ngrok / Nginx ──► Daphne ──► Django View ──► SQLite ──► HTML


  VIDEO CALL  (WebRTC via LiveKit)
  ──────────────────────────────────────────────────────────────────────────────
  Browser ──► Nginx /ws/ ──► ip_hash ──► Daphne ──► livekit-proxy ──► LiveKit SFU :7880
  Browser ──► /meetings/token/ ──► Django ──► JWT token ──► LiveKit room


  FACE ATTENDANCE  (WebSocket + AI)
  ──────────────────────────────────────────────────────────────────────────────
  Browser (webcam frame) ──► Nginx ──► Daphne ──► FaceAttendanceConsumer
                                                        │
                                                        ├── Camera Service (OpenCV)
                                                        ├── Face Recognition
                                                        └── mark present ──► SQLite


  CAMERA FEED  (RTSP → HLS)
  ──────────────────────────────────────────────────────────────────────────────
  IP Camera ──► RTSP/TCP ──► Camera Service :8001
                                    │
                                    ├── FFmpeg  -c:v copy  (zero re-encode)
                                    ├── HLS segments  (3-segment rolling window)
                                    └── Main App proxy ──► Browser


  BACKGROUND TASKS
  ──────────────────────────────────────────────────────────────────────────────
  Django ──► Celery ──► Redis (broker) ──► Worker ──► Recording / Analytics ──► SQLite
```
